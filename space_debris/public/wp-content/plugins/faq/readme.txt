Create a styled question that shows the answer with a toggle and hides the answer with a toggle.


== Installation ==

1. Upload the entire `faq` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu within the WordPress Dashboard.

You will now see the `faq` menu in your WordPress admin panel.

== Using faq ==

Using the faq plugin is very simple. You simply need to create a FAQ page or any page you desire and use the shortcode block to output the styled question. 
ex: [faq q="Q.What time is it?"] It's time for you to get a watch[/faq]

faq = is the opening and closing tags
q = "your question that you want to show" in quotes.

after the first square bracket block you have your specific answer. here you can ouput the answer to your question. simple as that.

have fun using faq plugin.
